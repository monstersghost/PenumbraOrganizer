Penumbra Organizer v0.1.0-alpha

Early alpha preview

What this app does today:
- Finds Penumbra.
- Scans installed mods in read-only mode.
- Shows the Penumbra state directory and mod library root.
- Lets you create proposed Penumbra folders in memory.
- Lets you organize selected mods into proposed folders.
- Supports protected mods, Undo, Redo, saved organizer sessions, and Review Changes.
- Can create an optional sanitized AI review package.

What this alpha does not do yet:
- It does not apply changes to Penumbra.
- It does not create verified backups.
- It does not roll back changes.
- It does not import AI proposals in the GUI yet.
- It does not move physical mod folders or edit game files.

How to open it:
1. Extract the ZIP.
2. Double-click PenumbraOrganizer.exe.
3. If Windows SmartScreen warns you, confirm the file came from the official GitHub Releases page before running it.

Basic use:
1. Click Find Penumbra.
2. Confirm the Penumbra state directory and mod library root.
3. Click Scan My Mods.
4. Open Organize.
5. Choose a starting strategy or Start Manually.
6. Create proposed folders.
7. Select mods and assign them to proposed folders.
8. Use Undo or Redo if needed.
9. Open Review Changes to inspect the proposal.

Saved organizer sessions:
- Click Save Session to save your current proposal.
- Click Resume Last Session after scanning the same library again.
- Sessions are app-owned proposal files only. They are not Penumbra files.

Closing safely:
- You can close the app at any time.
- This alpha does not write organization changes into Penumbra.

App-owned data:
- Saved sessions and exports are stored under:
  %LocalAppData%\PenumbraOrganizer\

Reporting a problem:
- Include the Penumbra Organizer version, Windows version, Penumbra version, what you clicked, and the plain-language error.
- Do not upload mod files, paid/private mods, full Penumbra configs, personal paths without redaction, or credentials.
