# Security Policy

## Supported versions

Only the latest public alpha is supported for security reports.

## Reporting a vulnerability

Do not disclose:

* paid or private mod assets
* credentials
* personal paths
* full Penumbra configurations
* API keys or tokens
* private exported inventories

For severe vulnerabilities, use GitHub's private vulnerability reporting if it is available for this repository.

For lower-risk issues, open a GitHub issue with a minimal sanitized description. Redact personal paths and do not attach mod files or full configuration directories.

## Scope

Penumbra Organizer should not modify game files, move physical mod folders, or operate on `.pmp` packages in the current alpha.
