# Penumbra Organizer v0.1.0-alpha

## Highlights

* first public alpha
* automatic Penumbra discovery
* read-only scanning
* visual manual organization foundation
* multiple organization strategies
* protected mods
* undo/redo
* session persistence
* Review Changes validation
* external AI package export
* portable self-contained Windows build

## Not yet available

* applying changes to Penumbra
* backup and rollback
* AI proposal import
* drag-and-drop
* collection editing
* `.pmp` handling

## Safety

This alpha does not write organization changes into Penumbra. It scans installed mods, lets users build in-memory proposals, and validates those proposals in Review Changes.
